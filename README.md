# DjangoVue
A boilerplate using Django and Vue.

## Installation 
#### Clone the repository`

#### Run
```python manage.py runserver 0.0.0.0:8000```

#### Install packages
```npm install```

#### Run client
```npm run dev```

#### Build production
```npm run build```


#### Test
  * Frontend: http://localhost:8080/
  * Backend: http://localhost:8000/
